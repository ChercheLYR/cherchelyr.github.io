Cherche~2021-04-012024-01-13

Definition 1:

Let XXX is a fixed set(or a Basic Space), and denote E:={E:E⊂X}\mathcal{E} := \left\{ E:E\subset X \right\}E:={E:E⊂X}. Call E\mathcal{E}E is a set class on XXX.

Definition 2:

Let E\mathcal{E}E is a set class on XXX, and if its elements satisfy:

∀E1,E2∈E\forall E_1,E_2 \in \mathcal{E}∀E1​,E2​∈E, have E1∪E2∈EE_1 \cup E_2 \in \mathcal{E}E1​∪E2​∈E and E1−E2∈EE_1-E_2 \in \mathcal{E}E1​−E2​∈E, then we called E\mathcal{E}E is a Ring on XXX and denote: R\mathcal{R}R.

In particular, if we have X∈RX \in \mathcal{R}X∈R, we called R\mathcal{R}R is a σ−\sigma-σ−Ring on XXX.

Definition 3:

Let E\mathcal{E}E is a set class on XXX, and define the mapping from E\mathcal{E}E to XXX:

μ:E→R\mu : \mathcal{E}\to \mathbb{R} μ:E→R

then, called μ\muμ is set function on E\mathcal{E}E.

Definition 4:

Let R\mathcal{R}R is a Ring on XXX, and we define the set function μ:R→R\mu : \mathcal{R}\to \mathbb{R}μ:R→R, and satisfied:

i>. μ(∅)=0\mu (\varnothing) = 0μ(∅)=0

ii>. (Non-negative) ∀E∈R\forall E \in \mathcal{R}∀E∈R, has μ(E)≥0\mu (E) \ge 0μ(E)≥0

iii>.(Additivity) En∈RE_n \in \mathcal{R}En​∈R, n=1,2,...n=1,2,...n=1,2,..., and ∀Ei∩Ej=∅(i≠j)\forall E_i \cap E_j=\varnothing (i \ne j)∀Ei​∩Ej​=∅(i​=j), have:

μ(⋃n=1∞En)=∑n=1∞μ(En)\mu \left(\bigcup_{n=1}^{\infty}E_n \right) =\sum_{n=1}^{\infty}\mu(E_n) μ(n=1⋃∞​En​)=n=1∑∞​μ(En​)

then, we called μ\muμ is a measure on R\mathcal{R}R.

Theorem

If R\mathcal{R}R is a Ring on XXX, and μ\muμ is a measure on R\mathcal{R}R, then it has:

  i>. (Limited additivity) If E1,E2,...,En∈RE_1,E_2,...,E_n \in \mathcal{R}E1​,E2​,...,En​∈R, and ∀Ei∩Ej=∅(i≠j)\forall E_i \cap E_j=\varnothing (i \ne j)∀Ei​∩Ej​=∅(i​=j),have:

μ(⋃i=1nEn)=∑i=1nμ(En)\mu \left(\bigcup_{i=1}^{n}E_n \right) =\sum_{i=1}^{n}\mu(E_n) μ(i=1⋃n​En​)=i=1∑n​μ(En​)

  ii>. (Monotonicity) If E1,E2∈RE_1,E_2 \in \mathcal{R}E1​,E2​∈R, and E1⊂E2E_1\subset E_2E1​⊂E2​, then

μ(E1)≤μ(E2)\mu(E_1) \le \mu(E_2) μ(E1​)≤μ(E2​)

μ(E2−E1)=μ(E2)−μ(E1)\mu(E_2-E_1)=\mu(E_2)-\mu(E_1) μ(E2​−E1​)=μ(E2​)−μ(E1​)

  iv>. (Secondary additivity) If En,E∈RE_n,E \in \mathcal{R}En​,E∈R, n=1,2,...n=1,2,...n=1,2,..., and E⊂⋃n=1∞EnE\subset \bigcup_{n=1}^{\infty}E_nE⊂⋃n=1∞​En​, then

μ(⋃n=1∞En)=lim⁡n→∞μ(En)\mu \left(\bigcup_{n=1}^{\infty}E_n \right) =\lim_{n\to \infty}\mu(E_n) μ(n=1⋃∞​En​)=n→∞lim​μ(En​)

μ(⋂n=1∞En)=lim⁡n→∞μ(En)\mu \left(\bigcap_{n=1}^{\infty}E_n \right) =\lim_{n\to \infty}\mu(E_n) μ(n=1⋂∞​En​)=n→∞lim​μ(En​)
