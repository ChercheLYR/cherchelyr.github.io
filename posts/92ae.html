  定义： 我们用 σ(n)\sigma(n)σ(n) 表示自然数 nnn 的所有因子之和，即

σ(n)=∑m∣nm\sigma(n) = \sum_{m|n}{m} σ(n)=m∣n∑​m

  (1). 若一个自然数 nnn , 做素因数分解：n=p1a1⋅⋅⋅pkakn = p_{1}^{a_1}\cdot\cdot\cdot p_{k}^{a_k}n=p1a1​​⋅⋅⋅pkak​​, 则有

σ(n)=p1a1+1−1p1−1⋅⋅⋅pkak+1−1pk−1\sigma(n) = \dfrac{p_{1}^{a_1+1} - 1}{p_1 - 1} \cdot\cdot\cdot \dfrac{p_{k}^{a_k+1} - 1}{p_k - 1} σ(n)=p1​−1p1a1​+1​−1​⋅⋅⋅pk​−1pkak​+1​−1​

证明：

  注意到

∏i=1k∑j=0αipij=∑(j1,⋅⋅⋅,jk)∈∏i=1k([0,αi]⋂N)p1j1⋅⋅⋅pkjk\prod_{i=1}^{k} {\sum_{j=0}^{ { {\alpha } _ {i} } } {p_{i}^{j} } } =\sum\limits_{\left( { {j}_{1} },\cdot \cdot \cdot ,{ {j}_{k} } \right)\in \prod_{ i=1 }^{k} {\left( \left[ 0,{ {\alpha }_{i} } \right]\bigcap N \right)} } { p_{1}^{ { {j}_{1} } } \cdot \cdot \cdot p_{k}^{ { {j}_{k} } } } i=1∏k​j=0∑αi​​pij​=(j1​,⋅⋅⋅,jk​)∈∏i=1k​([0,αi​]⋂N)∑​p1j1​​⋅⋅⋅pkjk​​

  而

σ(n)=p1a1+1−1p1−1⋅⋅⋅pkak+1−1pk−1=LHS=PHS=σ(n)\sigma(n) = \frac{p_{1}^{a_1 + 1} - 1}{p_1 - 1} \cdot\cdot\cdot \frac{p_{k}^{a_k+1} - 1}{p_k - 1} = LHS = PHS = \sigma(n) σ(n)=p1​−1p1a1​+1​−1​⋅⋅⋅pk​−1pkak​+1​−1​=LHS=PHS=σ(n)

  (2). 下面给出 σ(n)\sigma(n)σ(n) 的一个积分表达

σ(n)=∑m=1n∫0mcos⁡2nπ[x+1]mdx\sigma(n) = \sum_{m=1}^{n}\int_{0}^{m}\cos{\dfrac{2n\pi[x+1]}{m}}dx σ(n)=m=1∑n​∫0m​cosm2nπ[x+1]​dx

证明:

  我们考虑求和号里面的式子

  注意到, 利用 Euler 公式, 有

∑k=1mcos⁡2knπm=Re(∑k=1me2nkπim)\sum_{k=1}^{m}\cos{\dfrac{2kn\pi}{m}} = Re\left(\sum_{k=1}^{m}e^{\dfrac{2nk\pi i}{m}}\right) k=1∑m​cosm2knπ​=Re⎝⎜⎜⎛​k=1∑m​em2nkπi​⎠⎟⎟⎞​

∑k=1me2nkπim={e2nmπim−1e2nπim−1=0,m∤nm,m ∣ n\sum_{k=1}^{m}e^{\dfrac{2nk\pi i}{m}} = \left\{\begin{matrix} \dfrac{e^{\dfrac{2nm\pi i}{m}}-1}{e^{\dfrac{2n\pi i}{m}}-1} = 0,\quad m \nmid n\\ m, \quad m\ |\ n \end{matrix}\right. k=1∑m​em2nkπi​=⎩⎪⎪⎪⎪⎪⎨⎪⎪⎪⎪⎪⎧​em2nπi​−1em2nmπi​−1​=0,m∤nm,m ∣ n​
